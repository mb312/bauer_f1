🏎️ Custom F1-Project

A custom Formula 1 web application built with React that displays:
⏳ Countdown to the next race
🏁 Current driver standings
🏆 Constructor standings
📅 Full race calendar overview
📊 Data fetched from the Ergast API

🚀 Features
Live countdown to the next Formula 1 race
Current season overview
Driver standings
Constructor (team) standings
Full race calendar view
Dynamic menu navigation
API-driven content
Responsive UI

🛠️ Tech Stack
React
React Hooks (useState, useEffect)
React Router (for page navigation)
CSS / Tailwind / Styled Components (edit depending on what you used)
Fetch API / Axios
Data from the Ergast API via https://api.jolpi.ca/ergast/
Data from the OpenF1 API via https://api.openf1.org/v1/
